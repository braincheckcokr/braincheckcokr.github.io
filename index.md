## What We Do

**BrainCheck**는 플래시카드를 핵심으로 **언어 학습의 본질**을 다루는 플랫폼입니다.

| 듣기 | 말하기 |
|:---:|:---:|
| 소리 인식 + 정답 판별 | 생각 → 언어 변환 + 검증 |

우리는 이 과정을 **플래시카드**라는 단순한 인터페이스로 추상화합니다.

> **학습자가 항상 '가장 기억이 잘 되는 상태'에 머물도록 학습 흐름을 조정합니다.**

---

<h2 id="features">Features</h2>

- **이해도 추적** — 학습자의 이해도, 기억 유지율, 망각 속도를 지속적으로 분석
- **최적 타이밍** — 가장 효과적인 순간에 복습 자료 제공
- **즉각 피드백** — 발음, 정확성, 문장 완성도 실시간 평가

---

<h2 id="tech">Tech Stack</h2>

| 기술 | 설명 |
|---|---|
| **ML 추천 시스템** | 망각 패턴 분석 → 최적 학습 타이밍 선택 |
| **TTS** | 실제 대화 수준의 자연스러운 음성 생성 |
| **STT + 발화 평가** | 발음/정확성/완성도 분석 및 피드백 |
| **AI 콘텐츠 생성** | 텍스트/음성 → 플래시카드 자동 변환 |
| **멀티 플랫폼** | iOS, Android, macOS, Windows |

---

<h2 id="contents">Contents</h2>

BrainCheck에서 유저는 **학습자이자 제작자**입니다.

```
책/음성 입력 → AI 자동 생성 → 검수 → 공유/판매
```

- 누구나 **학습 덱**을 직접 제작
- 다른 유저에게 **공유 또는 판매**
- AI가 콘텐츠 자동 생성, 사용자가 검수 후 배포

## Mission

> **우리는 사람들이 원하는 것을 최소한의 시간으로 학습할 수 있도록 돕습니다.**

학습은 더 많은 시간을 투자하는 것이 아니라, **불필요한 시간을 제거**하는 것입니다.

---

<h2 id="principles">Principles</h2>

### 1. 완벽함은 덜어냄에서 온다

> "완벽함이란 더 이상 더할 것이 없을 때가 아니라 더 이상 뺄 것이 없을 때 이루어진다."
> — 생텍쥐페리

### 2. 학습은 '도달'의 과정이다

정보를 축적하는 것이 아니라, **특정 주제의 진실에 스스로 도달할 수 있게 되는 상태**입니다.

### 3. 쉬운 설명은 짧고, 전체를 보여준다

상대가 **"이해했다"**고 느끼게 하는 설명. 개별 정보가 아닌 **전체 구조와 전망**을 보여줍니다.
