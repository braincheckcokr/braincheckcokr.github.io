### AI 튜터

- 학습 중 질문에 친절하게 답변하는 AI 튜터
- 24시간 언제든 질문 가능
- 문법, 뉘앙스, 활용법 등 즉시 설명

### 발음 교정/체크

- 사용자 발음 실시간 검사 및 교정
- 원어민 발음과 비교 피드백
- 반복 연습으로 발음 개선

### 리그 시스템

- 경쟁 기반 동기부여 시스템
- 랭킹으로 학습 의욕 고취
- 주간/월간 리그 운영

### 학습 콘텐츠 생성/공유

- 사용자 직접 콘텐츠 제작 가능
- 다른 학습자와 공유
- 커뮤니티 기반 학습 자료 확장

### 단어 뉘앙스 설명

- 단순 뜻 외에 미묘한 의미 차이 설명
- 비슷한 단어 간 구분법
- 상황별 적절한 단어 선택 가이드

### 문법 설명

- 이해하기 쉬운 문법 설명
- 복잡한 문법도 간단하게 풀이
- 예문과 함께 문법 규칙 제시

- 구독료 기반 프리미엄 서비스
- 월간/연간 구독 옵션
- 전체 콘텐츠 무제한 접근
- 광고 제거 및 프리미엄 기능 포함
